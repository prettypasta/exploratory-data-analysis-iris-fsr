Iris Dataset Exploratory Data Analysis

This project presents an exploratory data analysis (EDA) of the Iris dataset using Python. The analysis focuses on understanding the relationships between different flower measurements, identifying data distributions, detecting outliers, and studying correlations between variables.

The project includes:

Data loading and preprocessing using Pandas and Scikit-learn
Descriptive statistical analysis
Data visualization with histograms, boxplots, scatter plots, pairplots, and heatmaps
Correlation analysis between quantitative variables
L2 normalization of numerical features
Interpretation of results and variable relationships
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Dataset

The Iris dataset contains morphological measurements of three flower species:

Setosa
Versicolor
Virginica

The analyzed eatures include:

Sepal length
Sepal width
Petal length
Petal width
